# SmartCRM – Backend

SmartCRM is a **smart customer relationship management tool** designed to enhance business communication through **automation and intelligence**.  
This repository contains the **Spring Boot backend** responsible for handling API logic, data processing, and integrations.

---

## Overview

The SmartCRM backend powers all business logic, including:

-  **Image Text Extraction** – Process and extract text from uploaded images.  
-  **Smart Email Generation** – Create AI-driven email drafts based on context.  
-  **Context Summarization** – Generate concise summaries of conversation history.  
-  **Email Activity Tracking** – Track and manage email-sending activity and status.  
-  *(Upcoming)* Bulk email automation and advanced analytics.

---

## ⚙ Tech Stack

- **Framework:** Spring Boot (Java 17+)  
- **Database:** PostgreSQL  
- **Build Tool:** Maven  
- **API Layer:** RESTful APIs  
- **Security:** Spring Security (JWT-based planned)  
- **AI Integration:** OpenAI API for text generation and summarization  
- **Version Control:** Git + GitHub
